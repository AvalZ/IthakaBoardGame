Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

* AndroidManifest.xml => IthakaBoardGame/src/main/AndroidManifest.xml
* assets/ => IthakaBoardGame/src/main/assets/
* libs/android-support-v4.jar => IthakaBoardGame/libs/android-support-v4.jar
* res/ => IthakaBoardGame/src/main/res/
* src/ => IthakaBoardGame/src/main/java/
